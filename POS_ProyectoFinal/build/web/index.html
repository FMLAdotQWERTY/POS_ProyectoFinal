POS: Proyecto Final


Esta es una aplicación web (prototipo) con la funcionalidad de emular un foro de películas, en el cual se pueden registrar usuarios, hacer una autorización mediante un login, y su vez el usuario puede contar con preferencias enlazadas a su cuenta. Las preferencias pueden ser: películas, genero de película, directores y actores.

La aplicación cuenta con un buscador que consume la API OMDb API: The Open Movie Database y muestra recomendaciones de películas dependiendo de las preferencias con las que cuenta el usuario.
